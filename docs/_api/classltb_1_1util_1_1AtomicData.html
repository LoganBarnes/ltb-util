Template Class AtomicData

  • Defined in File atomic_data.hpp

Class Documentation

template<typename T>
class ltb::util::AtomicData

Owns complex data that can be accessed in a thread safe way.

Example: 

struct MyComplexData {
    int thing1;
    bool thing2;
    std::vector<double> more_things;
    OtherStruct complex_thing;
};

ltb::util::AtomicData<MyComplexData> shared_data;

... Later, in different threads

shared_data.use_safely([] (MyComplexData& data) {
    // Do things with 'data' here:
    ...
});

...
