LTB_MAKE_WARNING(msg)

Macro used to auto-fill line and file information when creating a warning. 

ltb::util::Error error = LTB_MAKE_WARNING("Not so bad thing happened!");
assert(error.severity() == ltb::util::Error::Severity::Warning);
